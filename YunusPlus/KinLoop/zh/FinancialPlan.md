# Kin-Loop 財務計畫 (Financial Plan)

本計畫以金城鎮後浦示範區為基礎，預估首年運營預算。

## 1. 初期投資成本 (Startup Costs)
| 項目 | 預算 (NTD) | 說明 |
| :--- | :--- | :--- |
| **食品級循環杯採購** | 100,000 | 1,000 個高耐用度杯子（含金門限定設計）。 |
| **自動歸還箱/系統開發** | 120,000 | LINE Bot 租借系統與 3 座智慧感應歸還箱。 |
| **中央清洗站設備** | 180,000 | 專業商用洗碗機、高溫殺菌櫃、乾燥設備。 |
| **初期宣傳套件** | 30,000 | 合作店家立牌、教學影片。 |
| **總計** | **430,000** | **初期投資額** |

## 3. 預估收入 (Revenue Forecast)
| 項目 | 預估每月收入 (NTD) | 基準 |

## 4. 財務平衡分析 (Sustainability)
- **運營穩定性**：每月 1.5 萬杯的流轉率（平均每日 500 杯）即可維持基本運營平衡。
- **擴展潛力**：
    - 若擴展至金門機場與碼頭，租借數量預計成長 3 倍。
    - 減少政府垃圾清運成本估計每年達 50 萬以上，可爭取公共服務標案。
- **獎金運用**：若獲獎，將用於「身障者洗滌專線培訓」，並購置更先進的杯子追蹤晶片。
