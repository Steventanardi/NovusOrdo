# Kin-Loop (金門減塑小圈計畫)
### 金門景點共享循環容器服務

## 1. 社會問題 (Social Problem)
金門觀光商機帶動大量一次性塑料、紙膠杯垃圾。由於金門本身缺乏焚化設備，垃圾必須海運回台灣處理，運輸成本高且碳排量巨大。觀光客在後浦老街、沙美老街購買手搖飲後，垃圾隨處丟棄影響古蹟景觀。

## 2. 解決方案與原型 (Solution & Prototype)
**解決方案**：
建立「金門專屬循環杯系統」。在金城鎮（Houpu District）手搖店密集的 500 公尺範圍內，提供循環杯租借服務。觀光客借用、飲用、歸還，由社會企業統一回收清洗。

**原型設計**：
選定「Jincheng Houpu Street」作為示範區，合作門市 5-10 間，設置自動歸還機（KIOSK）。

## 3. 工作流程 (Workflow)
1. **借用 (Borrow)**：顧客掃碼 LINE 官方帳號支付壓金（或免壓金綁定），領取循環杯。
2. **飲用 (Consume)**：顧客在老街散步，享用飲料。
3. **歸還 (Return)**：顧客至任何合作店家或路口設置的「自動歸還箱」投遞。
4. **回收與清洗 (Cleansing)**：由特約勞動團隊（僱用在地身心障礙者或單親家長）進行專業醫療級高溫清洗。
5. **再次投放 (Redistribution)**：每日定時配送乾淨杯子回合作店家。

## 4. 實施細節 (How to do it)
- **階段一（第一至二個月）**：與「金城鎮老街發展協會」合作，談妥 5 家指標手搖飲店。
- **階段二（第三個月）**：設計金門特色杯款（如：風獅爺圖騰），增加收藏感與拍照意願。
- **階段三（第四至六個月）**：啟動「1000 杯試行計畫」，目標單月減少 10,000 個一次性杯子。

## 5. 商業模式 (Business Model)
- **收入來源**：
    - 店家手續費（每杯租金 2-3 元，低於店家採購紙杯+塑膠蓋成本）。
    - 杯身廣告贊助（在地名產品、觀光導覽）。
    - 壓金孳息或逾期未還費用。
- **社會型企業原則**：
    - 利潤用於改善洗滌站設備與提高基層勞工薪資。
    - 與尤努斯基金會合作，推廣「三零目標」認證標誌。
