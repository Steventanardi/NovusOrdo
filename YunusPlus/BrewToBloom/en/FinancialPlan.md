# Brew-to-Bloom Financial Plan (EN) - Realistic Pilot Scale

This plan assumes a small-scale pilot managing ~150kg of grain per day.

## 1. Initial Investment (CAPEX)
| Item | Cost (NTD) | Rationale |
| :--- | :--- | :--- |
| **Container & Refit** | 180,000 | Second-hand 20ft container + insulation + ventilation. |
| **Machinery** | 70,000 | Small shredder, mixer, and pelletizer. |
| **IoT & Sensors** | 20,000 | Temp/Humidity monitoring (Crucial for BSF health). |
| **Permits & Misc** | 30,000 | Business registration and environmental testing. |
| **Total** | **300,000** | |

## 2. Monthly Working Capital (OpEx)
| Item | Cost (NTD) | Description |
| :--- | :--- | :--- |
| **Labor** | 50,000 | 2 Local part-timers (Seniors/Youth) at 25k each. |
| **Utilities** | 6,000 | Electricity for ventilation and water. |
| **Logistics** | 12,000 | Fuel and maintenance for one second-hand truck. |
| **Total** | **68,000** | |

## 3. Revenue Projections (Conservative)
| Item | Revenue (NTD) | Logic |
| :--- | :--- | :--- |
| **Compost Sales** | 35,000 | 350 bags (5kg) @ 100 NTD. |
| **Larvae Sales** | 25,000 | As organic feed for local poultry farmers. |
| **Total** | **60,000** | |

## 4. Short-term Sustainability
- **Monthly Gap**: -8,000 NTD initially.
- **Sustainability Strategy**: The gap is covered by the Yunus Innovation Award prize money for the first 12 months.
- **Scale Up**: Once the model proves its fertilizer quality, we can charge a "Waste Disposal Fee" to larger distilleries (approx. 15,000 - 20,000 NTD/month), which brings the business to a **Net Profit of 7,000 - 12,000 NTD**.
- **Social Impact**: Zero dividend strategy. The surplus is used to sponsor soil analysis for smallholder farmers.
