# Rencana Keuangan Brew-to-Bloom (ID) - Skala Pilot Realistis

Rencana ini mengasumsikan pilot skala kecil yang mengelola ~150kg ampas per hari.

## 1. Investasi Awal (CAPEX)
| Item | Biaya (NTD) | Alasan |
| :--- | :--- | :--- |
| **Modifikasi Kontainer** | 180.000 | Kontainer bekas 20 kaki + isolasi + ventilasi. |
| **Mesin** | 70.000 | Penghancur kecil, mixer, dan pembuat pelet. |
| **IoT & Sensor** | 20.000 | Pemantauan Suhu/Kelembaban (Sangat penting bagi kesehatan BSF). |
| **Izin & Lain-lain** | 30.000 | Registrasi bisnis dan pengujian lingkungan. |
| **Total** | **300.000** | |

## 2. Modal Kerja Bulanan (OpEx)
| Item | Biaya (NTD) | Deskripsi |
| :--- | :--- | :--- |
| **Tenaga Kerja** | 50.000 | 2 Pekerja lokal paruh waktu (Lansia/Pemuda) @ 25rb. |
| **Utilitas** | 6.000 | Listrik untuk ventilasi dan air. |
| **Logistik** | 12.000 | Bahan bakar dan perawatan truk bekas. |
| **Total** | **68.000** | |

## 3. Proyeksi Pendapatan (Konservatif)
| Item | Pendapatan (NTD) | Logika |
| :--- | :--- | :--- |
| **Penjualan Kompos** | 35.000 | 350 kantong (5kg) @ 100 NTD. |
| **Penjualan Larva** | 25.000 | Sebagai pakan organik untuk peternak unggas lokal. |
| **Total** | **60.000** | |

## 4. Keberlanjutan Jangka Pendek
- **Celah Bulanan**: -8.000 NTD pada awalnya.
- **Strategi Keberlanjutan**: Celah ini ditutup oleh uang hadiah Yunus Innovation Award untuk 12 bulan pertama.
- **Peningkatan Skala**: Setelah model membuktikan kualitas pupuknya, kami dapat membebankan "Biaya Pembuangan Limbah" ke penyulingan yang lebih besar (sekitar 15.000 - 20.000 NTD/bulan), yang membawa bisnis ke **Laba Bersih 7.000 - 12.000 NTD**.
- **Dampak Sosial**: Strategi dividen nol. Surplus digunakan untuk mensponsori analisis tanah bagi petani kecil.
