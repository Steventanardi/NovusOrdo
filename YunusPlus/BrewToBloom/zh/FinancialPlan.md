# Brew-to-Bloom 財務計畫 (Financial Plan)

本計畫以小型規模（Pilot Scale）為基礎，預估首年運營預算與收支。

## 1. 初期投資成本 (Startup Costs)
| 項目 | 預算 (NTD) | 說明 |
| :--- | :--- | :--- |
| **貨櫃設備改造** | 150,000 | 20 呎貨櫃、通風、室溫控制設備。 |
| **黑水虻/菌種入料** | 20,000 | 引進高效發酵菌種及首批蟲苗。 |
| **粉碎與包裝設備** | 80,000 | 小型粉碎機、簡易封口包裝機。 |
| **雜項與行銷** | 50,000 | 標籤設計、在地宣傳。 |
| **總計** | **300,000** | **初期資本支出 (CAPEX)** |

## 2. 每月營運支出 (OpEx)
| 項目 | 費用 (NTD) | 說明 |
| :--- | :--- | :--- |
| **人力工資** | 50,000 | 兼職操作員 2 名（在地二度就業）。 |
| **材料與水電** | 5,000 | 發酵電力消耗及輔料採購。 |
| **物流运输** | 10,000 | 金門島內酒糟收集與肥料運送。 |
| **每月總計** | **65,000** | |

## 3. 預估收入 (Revenue Forecast)
| 項目 | 預估每月收入 (NTD) | 基準 |
| :--- | :--- | :--- |
| **有機肥料銷售** | 30,000 | 每包 5kg/100 元，每月銷售 300 包。 |
| **飼料蟲銷售** | 20,000 | 供應當地 3 家養雞場。 |
| **ESG/廢棄物委託** | 15,000 | 與酒廠收取低額環保處理費。 |
| **每月總計** | **65,000** | |

## 4. 財務平衡分析 (Sustainability)
- **損益平衡點 (Breakeven)**：約在運營第 6 個月達到月度收支平衡。
- **社會影響力再投資**：
    - 多餘利潤將用於增設「自動化包裝線」。
    - 預計第 18 個月可收回初期設備投資。
- **獎金運用**：若獲得尤努斯創新獎金，將全數投入「金門在地小農土壤改良補助金」，免費提供肥料給 50 戶困苦農戶。
