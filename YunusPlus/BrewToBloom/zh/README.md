# Brew-to-Bloom (酒糟轉身計畫)
### 金門高粱酒糟循環經濟社會企業

## 1. 社會問題 (Social Problem)
金門作為高粱酒之鄉，每日產生大量酒糟（Distillery Grains）。除了大部分由大廠回收外，民間小型酒廠的酒糟處理與環境成本日益增高。同時，金門長期種植高粱導致土壤貧瘠、有機質不足。本計畫旨在建立一個「小型村落級」的酒糟轉化站。

## 2. 解決方案與原型 (Solution & Prototype)
**解決方案**：建立「微型酒糟轉化站」，透過黑水虻（BSF）或快速發酵技術，將新鮮酒糟轉化為：
1. **有機肥料**：改善當地高粱與蔬菜田土壤。
2. **高蛋白飼料**：提供給當地養雞戶，打造「酒糟油雞」品牌。

**原型設計**：使用 20 呎貨櫃改裝的行動發酵艙，可在村落間部署，初期選擇金寧鄉或金城鎮試辦。

## 3. 工作流程 (Workflow)
1. **收集 (Input)**：固定時段向小型酒廠或農村小規模收集剩餘酒糟及落葉碎屑。
2. **處理 (Processing)**：在貨櫃箱內進行控溫發酵（黑水虻分解需 10-14 天）。
3. **成品 (Output)**：產出黑水虻糞（高品質有機肥）與成熟幼蟲（乾製或冷凍）。
4. **配送 (Logistics)**：將肥料回饋給簽約農戶，幼蟲賣給在地蛋雞場。
5. **數據監測**：紀錄碳減排數據，作為 ESG 報告基礎。

## 4. 實施細節 (How to do it)
- **階段一（第一至三個月）**：與 2 家在地小型酒廠簽署實驗合作協議。
- **階段二（第四至六個月）**：部署第一座「貨櫃轉化艙」，日處理量控制在 100-200kg 以內。
- **階段三（第七個月）**：申請台灣尤努斯創新獎，並尋求金門縣政府環保局或農業處補助。

## 5. 商業模式 (Business Model)
- **收入來源**：
    - 訂閱制肥料銷售（農民）。
    - 黑水虻幼蟲/飼料銷售（養雞戶）。
    - 企業 ESG 數據服務費。
- **社會型企業原則**：利潤 100% 用於擴大規模及聘僱在地二度就業婦女/退休長者。
