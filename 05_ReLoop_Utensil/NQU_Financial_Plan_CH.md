# 📊 ReLoop 財務對比與試點計畫（金門大學 NQU 小規模 - 100 餐）

本文件基於 Huang et al. (2026) 研究數據，針對每日 100 餐規模的 **ReLoop** 循環餐具試點計畫提供財務分析。

---

## ⚖️ 財務對比：一次性餐具 vs. ReLoop

### 1. 一次性餐具方案（現狀）
台灣標準外帶組合（紙餐盒 + 塑膠蓋 + 免洗筷 + 塑膠湯匙 + 提袋）的估計成本。

| 項目 | 成本 (NTD) | 備註 |
| :--- | :--- | :--- |
| 紙餐盒 + 蓋子 | 2.50 | 批發價 |
| 免洗筷（含包裝） | 0.50 | 標準規格 |
| 塑膠湯匙 | 0.30 | 大量採購 |
| 紙袋/塑膠袋 | 0.20 | 分發成本 |
| **每餐總計** | **3.50** | **100% 廢棄物** |

**每日成本 (100 餐)：NT$ 350**  
**年度成本 (300 天)：NT$ 105,000**

---

### 2. ReLoop 循環餐具方案（預測 - 100 組）

#### A. 初始投資 (CAPEX)
針對每日 100 餐的規模，需要儲備 250 組單元（循環中 + 清洗中 + 庫存）。

| 項目 | 單價 (NTD) | 總成本 (NTD) |
| :--- | :--- | :--- |
| 循環餐盒組 | 120 | 30,000 |
| 不鏽鋼餐具組 | 30 | 7,500 |
| 小型除菌處理機 | 60,000 | 60,000 |
| 智慧回收桶 (1 座) | 20,000 | 20,000 |
| **初始總投資** | | **117,500** |

#### B. 單次使用營運成本 (OPEX)
在極小規模下，由於台灣基本時薪固定，人力效率成為最大的財務挑戰。

| 支出類別 | 成本 (NTD) | 邏輯 |
| :--- | :--- | :--- |
| **高溫除菌（熱能 + 藥劑）** | **1.10** | **電力 (0.044kWh) @ 82°C** + 水費 + 洗劑 |
| **人力成本（回收與清洗）** | **1.90** | **每日 1 小時 @ NT$190/時** (基本時薪) / 100 組 |
| 損耗/更換基金 | 0.40 | 每次循環 1% 損耗率計 |
| **每餐總計** | **3.40** | **僅節省 NT$0.10/餐** |

**每日成本 (100 餐)：NT$ 340**  
**年度成本 (300 天)：NT$ 102,000 (+ CAPEX)**

---

## 📈 盈虧平衡分析

*   **每餐節省**: NT$ 3.50 (一次性) - NT$ 3.40 (合規營運) = **NT$ 0.10 節省**。
*   **需回收總投資**: NT$ 117,500。
*   **回本時間（以每日 100 餐計）**: **~39 年**（財務回收期）。

> [!WARNING]
> **規模化的挑戰**: 在每日僅 100 餐的規模下，人力的固定成本（基本時薪）占據了營運預算的 55%。若要達成 2-3 年內回本，系統必須 **擴大規模至每日 500+ 餐**，或利用 **共用人力資源**（例如由現有的餐廳員工兼職操作）。

## 🛡️ 營運品質控管與資產管理

### 1. 維持除菌狀態
*   **FIFO (先進先出)**: 優先使用存放久的餐具。
*   **UV-C 儲存**: 餐具乾燥後儲存於封閉紫外線殺菌櫃。

### 2. 追蹤資產壽命
*   **QR 識別**: 每個餐盒均有唯一 QR Code。
*   **自動淘汰**: 使用達 **500 次**後，送往 BSF 塑料回收循環。

## 🚀 NQU 小規模試點計畫（第一階段）

1.  **辦公室訂閱服務**: 鎖定教職員辦公室，提供「零廢棄午餐」訂閱方案。
2.  **人力優化**: 與金大總務處合作，將清洗任務整合至現有的餐飲後勤體系。
