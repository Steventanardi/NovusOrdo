# 📊 ReLoop 財務對比與試點計畫（金門大學 NQU 試點規模 - 300 餐）

本文件根據每日 300 餐的規劃，為金門大學 (NQU) 的 **ReLoop** 循環餐具試點計畫提供詳細財務成本分析與營運策略。

---

## ⚖️ 財務對比：一次性餐具 vs. ReLoop

### 1. 一次性餐具方案（現狀）
台灣標準外帶組合（紙餐盒 + 塑膠蓋 + 免洗筷 + 塑膠湯匙 + 提袋）的估計成本。

| 項目 | 成本 (NTD) | 備註 |
| :--- | :--- | :--- |
| 紙餐盒 + 蓋子 | 2.50 | 批發價 |
| 免洗筷（含包裝） | 0.50 | 標準規格 |
| 湯匙與提袋 | 0.50 | 消耗品 |
| **每餐總計** | **3.50** | **100% 廢棄物** |

**每日成本 (300 餐)：NT$ 1,050**  
**年度成本 (300 天)：NT$ 315,000**

---

### 2. ReLoop 循環餐具方案（預測 - 300 組）

#### A. 初始投資 (CAPEX)
針對每日 300 餐規模，需儲備 750 組單元（循環中 + 清洗中 + 庫存）。

| 項目 | 單價 (NTD) | 總成本 (NTD) |
| :--- | :--- | :--- |
| 循環餐盒組 | 120 | 90,000 |
| 不鏽鋼餐具組 | 30 | 22,500 |
| 工業級掀罩式洗碗機 | 95,000 | 95,000 |
| 智慧回收站 (2 座) | 30,000 | 60,000 |
| **初始總投資** | | **267,500** |

#### B. 單次使用營運成本 (OPEX)
基於台灣基本時薪與水電費率計算。

| 支出類別 | 成本 (NTD) | 邏輯 |
| :--- | :--- | :--- |
| **除菌能源費** | **0.40** | 水費 + 電費 (0.044kWh/組) |
| **場地租金** | **0.50** | 餐廳角落租金 NT$4,500/月 |
| **物流維運 (LINE)** | **0.10** | 電動餐車充電 + LINE API 規費 |
| **人力成本 (3小時/日)** | **1.90** | 3 小時 @ NT$190/時 / 300 組 |
| 損耗基金 | 0.40 | 每次循環 1% 損耗計 |
| **每餐總計** | **3.30** | **較一次性節省 NT$0.20/餐** |

**每日成本 (300 餐)：NT$ 990**  
**年度成本 (300 天)：NT$ 297,000 (+ CAPEX)**

---

## 📈 盈虧平衡分析（尤努斯模式）

*   **基礎節省**: 每餐僅節省 NT$0.20，回收投資時間較長。

> [!TIP]
> **商業模式轉向**: ReLoop 採集 **「環保服務附加費」 NT$10/餐**。
> *   單日淨利 (含附加費): **NT$ 2,010/日**。
> *   **投資回收期**: 僅需 **133 天**即可回本（約半個學年）。

## 🛡️ 營運品質控管與資產管理

### 1. 維持除菌狀態
*   **FIFO**: 透過 **LINE 群組庫存追蹤** 確保先進先出。
*   **UV-C 儲存**: 餐具儲存於紫外線殺菌櫃。

### 2. 追蹤資產壽命
*   **QR 掃描**: 維運人員於 **LINE 群組** 回報循環狀態。
*   **除役**: 達到 **500 次**循環後，送至 BSF 黑水虻農場製成養成盆。

## 🚀 NQU 指標性試點計畫

1.  **LINE 群組管理**: 
    *   **「ReLoop 維運群」**: 工讀生回報滿桶、機器狀態。

2.  **場地優化**: 與金大學生會合作，共享辦公空間作為洗條 Hub 以降低租金負擔。
