# 📊 ReLoop Perbandingan Finansial & Rencana Pilot (Skala Kecil NQU - 100 Porsi)

Dokumen ini menyajikan analisis keuangan berbasis data untuk layanan peralatan makan pakai ulang **ReLoop** di **Universitas Nasional Quemoy (NQU)**, yang disesuaikan untuk pilot 100 porsi per hari.

---

## ⚖️ Perbandingan Finansial: Sekali Pakai vs. ReLoop

### 1. Skenario Sekali Pakai (Status Quo)
Estimasi biaya untuk satu set makanan bawa pulang standar di Taiwan (Kotak kertas + tutup plastik + sumpit kayu + sendok plastik + kantong kertas).

| Item | Biaya (NTD) | Keterangan |
| :--- | :--- | :--- |
| Kotak Makan Kertas + Tutup | 2,50 | Harga grosir |
| Sumpit Kayu (bungkus) | 0,50 | Kualitas standar |
| Sendok Plastik | 0,30 | Pembelian massal |
| Kantong Kertas/Plastik | 0,20 | Biaya distribusi |
| **Total per Porsi** | **3,50** | **100% Sampah** |

**Biaya Harian (100 porsi): NT$ 350**  
**Biaya Tahunan (300 hari): NT$ 105.000**

---

### 2. Skenario ReLoop Pakai Ulang (Proyeksi - 100 Unit)

#### A. Investasi Awal (CAPEX)
Untuk 100 porsi per hari, dibutuhkan cadangan 250 unit (sirkulasi + pencucian + penyimpanan).

| Item | Biaya Satuan (NTD) | Total Biaya (NTD) |
| :--- | :--- | :--- |
| Set Wadah Pakai Ulang | 120 | 30.000 |
| Set Alat Makan Stainless Steel | 30 | 7.500 |
| Unit Sterilisasi Menengah | 60.000 | 60.000 |
| Bin Pengumpulan Pintar (1) | 20.000 | 20.000 |
| **Total CAPEX Awal** | | **117.500** |

#### B. Biaya Operasional per Pemakaian (OPEX)
Pada skala kecil ini, efisiensi tenaga kerja menjadi tantangan utama karena upah minimum per jam yang tetap.

| Kategori Biaya | Biaya (NTD) | Logika |
| :--- | :--- | :--- |
| **Sterilisasi (Panas + Kimia)** | **1,10** | **Listrik (0,044kWh) @ 82°C** + Air + Sanitizer |
| **Tenaga Kerja (Koleksi & Cuci)** | **1,90** | **1 jam/hari @ NT$190/jam** (Upah Min) / 100 unit |
| Dana Kehilangan/Penggantian | 0,40 | Tingkat kehilangan 1% per siklus |
| **Total per Porsi** | **3,40** | **Hemat NT$0,10 per porsi** |

**Biaya Harian (100 porsi): NT$ 340**  
**Biaya Tahunan (300 hari): NT$ 102.000 (+ CAPEX)**

---

## 📈 Analisis Titik Impas (Break-Even)

*   **Penghematan per Porsi**: NT$ 3,50 (Sekali pakai) - NT$ 3,40 (OPEX Layak Upah) = **NT$ 0,10 hemat**.
*   **Total CAPEX yang Harus Pulih**: NT$ 117.500.
*   **Waktu untuk Impas (pada 100 porsi/hari)**: **~39 Tahun** (Payback finansial).

> [!WARNING]
> **Tantangan Skala**: Pada 100 porsi/hari, biaya tenaga kerja (upah min) memakan 55% dari anggaran operasional. Untuk mencapai keberlanjutan finansial dalam 2-3 tahun, sistem harus **ditingkatkan ke 500+ porsi/hari** atau menggunakan **staffing bersama** (misalnya staf kantin yang sudah ada).

---

## 🛡️ Pengendalian Mutu Operasional & Manajemen Aset

### 1. Menjaga Sterilitas
*   **FIFO**: Gunakan stok terlama terlebih dahulu.
*   **Penyimpanan UV-C**: Simpan dalam kabinet tertutup dengan sinar UV-C.

### 2. Melacak Masa Pakai Aset
*   **Log QR**: Setiap wadah memiliki kode QR unik.
*   **Pensiun Otomatis**: Pada **500 pemakaian**, unit dipensiunkan ke loop daur ulang plastik BSF.

## 🚀 Rencana Pilot Skala Kecil NQU (Fase 1)

1.  **Layanan Subskripsi**: Kantor Departemen berlangganan NT$1.000/bulan untuk 20 porsi harian "Zero-Waste".
2.  **Strategi Tenaga Kerja**: Integrasi dengan staf NQU yang sudah menangani logistik kantin.
