## 🚨 背景：台灣電網仍有 70% 仰賴化石燃料——宿舍是隱形的能源黑洞

台灣於 2025 年初完成核電除役，目前能源結構約為**天然氣 50%、燃煤 31%、再生能源僅約 19%**。電力碳排係數為 **0.474 kg CO₂/kWh**（台電 2024 年官方數據）。宿舍內節省的每一度電，都直接減少了以化石燃料為主的電網排放。

大學宿舍的能源問題尤為特殊：
- **學生繳交定額或補貼電費** → 幾乎沒有節電的經濟誘因。
- **「隱形」耗電** — 睡覺時未關的冷氣、持續插著的充電器、老舊電器 24 小時運轉。
- **缺乏即時回饋** — 學生看不到自己的用電量，自然無從改變行為。

台灣大學的研究確認：**用電量最高的前 20% 學生，消耗了宿舍大部分的電力**。一套以行為引導為核心的即時監控系統，可以在不改造建築硬體的情況下，實現 **10–20% 的節電效果**。

## 🏛️ 一、 社會問題分析

### ⚠️ 看不見的浪費 — 宿舍能源揮霍的結構性成因

| 問題層面 | 說明 | 規模 |
| :--- | :--- | :--- |
| **補貼電費機制** | 多數台灣大學宿舍電費低於市場價格，排除節電誘因 | 全台 300+ 所大學 |
| **集體行動困境** | 個別學生感到無力 — 「我一個人省不了什麼」 | 台大宿舍即有 8,500 名住宿生 |
| **隱形待機耗電** | 充電器、延長線、老舊冰箱、冷氣待機：各自小量，合計龐大 | 估計佔住宅總負載的 15–25% |
| **缺乏回饋機制** | 沒有即時顯示 → 沒有行為改變 → 浪費月月循環 | 行為經濟學研究證實 |
| **老舊設備基礎** | 10–15 年以上的宿舍老舊冷氣，耗電量比現代變頻機高 30–50% | 適用於台灣 80%+ 公立大學宿舍 |

**台灣各大學現況**：
- **台灣大學（NTU）**：約 8,500 名宿舍住宿生
- **清華大學（NTHU）**：約 5,000 名學生分布於 18 棟宿舍
- **成功大學（NCKU）**：台南校區數千名住宿生
- 全台估計 **20 萬名以上**學生生活在大學宿舍
- 宿舍平均用電：**9–24 度/人/月**（年 108–288 度/人），重度使用者為中位數的 3–5 倍

### 基線試算：一層宿舍的浪費量

> 一層 20 間宿舍 × 2 人 × 平均 150 kWh/月 = **6,000 kWh/月**
> 按台電住宅電費均價（約 NT$4.5/度）計算：**月電費約 NT$27,000**
> 若節電 15%：**每月省下 NT$4,050** — 對學生而言是相當可觀的金額。

## 🚀 二、 公司目標與解決方案

> **「看不見的耗電，就省不了電。WattWatch 讓隱形的浪費現形，讓節電變成獎勵。」**

### 願景
到 2030 年，在台灣 **500 棟大學宿舍樓**中部署 WattWatch，平均降低宿舍用電量 **15%**——每年減少 **12 萬噸 CO₂** ——並透過「共享效益」模式，讓節省的電費直接回饋給學生。

### 解決方案：WattWatch = IoT 硬體 + SaaS 儀表板 + 共享節電效益引擎

#### 第一層 — WattWatch 智慧插座（硬體）
低成本、專為節電監控設計的裝置：
- **主控晶片**：Tuya 相容 BK7231 芯片組（量產成熟；1,000 片以上 BOM 約 $4–6 USD）
- **電量計量 IC**：BL0942 能源監控晶片——即時測量瓦特、度數、電壓、電流
- **連線方式**：2.4GHz Wi-Fi（使用現有宿舍網路，無須新建設施）
- **外型**：符合台灣 CNS 三孔插座標準；機身輕巧，附 LED 狀態指示燈
- **數據推送**：每 10 秒一次 → WattWatch 雲端 → 儀表板
- **目標售價**：NT$350–450/個（低於 NT$500 = 學生不考慮的衝動消費門檻）

#### 第二層 — WattWatch 儀表板（SaaS 網頁版 + LINE Bot）
即時、美觀、具社交設計：

| 功能名稱 | 運作方式 |
| :--- | :--- |
| **我的房間** | 即時瓦數、每日/每月用電度數、即時換算新台幣金額 |
| **樓層比較** | 匿名同儕比較——「你的房間用電量比樓層平均低 20%」 |
| **電器偵探** | AI 辨識用電模式：「你的冷氣似乎每晚開 6 小時——調高至 26°C 可每月省 NT$180」 |
| **綠能積分** | 遊戲化週積分：連續節電可獲點數，可兌換校園商店優惠 |
| **省電存錢罐** | 即時更新本月樓層/棟整體節電金額累積 |
| **碳排計數器** | 本週減少的 kg CO₂ 排放，顯示為「相當於停開燃煤電廠 X 分鐘」 |

#### 第三層 — 共享節電效益引擎（商業模式核心）
尤努斯模式的核心創新：**WattWatch 只有在學生真正存到錢時，才會收費**。

```
建立用電基線（前 1–2 個月數據）
↓
每個月：對比實際帳單與 AI 預測基線
↓
節電效益 = （基線用電量 − 實際用電量）× 台電電費單價
↓
分配：80% 歸宿舍/學生/學校 | 20% = WattWatch 服務費
```

**以一層宿舍（40 名學生）為例**：
- 基線：6,000 kWh/月 → 電費 NT$27,000
- 使用 WattWatch 後：5,100 kWh/月 → 電費 NT$22,950
- **節電效益：NT$4,050**
- 學校/學生保留：**NT$3,240**
- **WattWatch 收取：NT$810/層/月** — 學校零前期成本

## 📊 三、 商業模式與創新性

### 為什麼「共享節電效益」是最完美的尤努斯模式

| 傳統 SaaS 訂閱 | WattWatch 共享效益模式 |
| :--- | :--- |
| 無論績效皆收取月費 | **沒有節電效益則零收費** |
| 學校承擔全部財務風險 | **風險共擔 — WattWatch 有績效才有收入** |
| 學生沒有直接利益 | **學生在「省電存錢罐」中看到直接獎勵** |
| 預算拮据的學校難以說服 | **零前期成本，保證正投資報酬率** |

### 收益模型（以一棟 100 間宿舍為例）

| 情境 | 每月節電效益 | WattWatch 20% 服務費 |
| :--- | :--- | :--- |
| 保守估計（節電 10%） | NT$27,000 | **NT$5,400/月** |
| 目標水準（節電 15%） | NT$40,500 | **NT$8,100/月** |
| 樂觀估計（節電 20%） | NT$54,000 | **NT$10,800/月** |

### 單位經濟

| 項目 | 成本 |
| :--- | :--- |
| 硬體（WattWatch 插座，1,000 個起訂量） | NT$180–220/個（BOM + 組裝） |
| 每間宿舍安裝數量（1–2 個） | NT$360–440/間 |
| SaaS 基礎設施（雲端、儀表板、LINE Bot） | NT$5,000/月固定成本（服務所有棟） |
| 每棟宿舍（200 個插座）投資回收期 | **< 3 個月**（從開始收取服務費起算） |

### 創新：「電器護照」功能

每個插座追蹤電器的能源特徵碼，WattWatch AI 建立**全台匿名化電器能效資料集**：
- 「你的冷氣型號（2012 年大金舊機）平均耗電 1.8 kW——比現代機種高 40%」
- 學校設施管理部門可用此數據，優先汰換效益最高的老舊電器
- 政府採購部門可用此數據訂定最低能效標準

此**數據即服務（DaaS）**層（銷售給大學設施部門及政府能源主管機關），成為高利潤的 B2G 收益來源。

## 📈 五、 財務永續與策略規劃

### 「零風險大學提案」的市場策略

WattWatch 的市場進入策略，旨在消除每一個異議：

| 學校疑慮 | WattWatch 的回應 |
| :--- | :--- |
| 「我們沒有預算」 | 零前期成本 — 試點期間 WattWatch 免費安裝硬體 |
| 「需要 IT 部門審核」 | 使用現有 Wi-Fi；無須安裝伺服器；數據存放於台灣區雲端 |
| 「學生不會配合」 | LINE 整合 — 無須下載 App；同儕比較在學生社群中具天然病毒式傳播力 |
| 「如何驗證節電效益？」 | 量測驗證（M&V）協議 — 以實際台電帳單對比 AI 預測基線 |

### 策略路線圖

- **第一年（試點）**：在 1 所大學（目標：清華或台大）的 3 棟宿舍樓部署。600 個智慧插座。建立基線並驗證 12–18% 節電率。取得 USR（大學社會責任）報告認可。
- **第二年**：透過大學永續發展辦公室合作，擴展至 10 所大學。部署 5,000 個插座。推出「電器護照」B2G 數據產品，對接教育部。
- **第三年**：取得教育部「綠色校園計畫」的全國大學招標案。50+ 所大學。
- **第五年**：**500 棟宿舍樓**安裝完畢。**10 萬個智慧插座**運作中。每年共享效益收入達 NT$4,000 萬以上。授權至東南亞大學市場（印尼、越南、菲律賓——相同的宿舍基礎設施困境）。

### 「學生能源大使」社會影響力層次

每棟加入的宿舍樓指派 2 名學生大使：
- 管理樓層節電競賽
- 從 WattWatch 服務費分潤中獲得 NT$500/月工讀工時補貼
- 接受能源識讀培訓——可進入職場的永續技能認證
- 建立學生間的同儕節電問責文化
