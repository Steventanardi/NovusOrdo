# Budget Proposal: Energy Tile Pilot Project (Taipei MRT)

This budget estimate is based on typical costs for a **20-square-meter (approx. 215 sq ft)** pilot installation using high-efficiency electromagnetic induction tiles (e.g., Pavegen-style).

## Project Scope
*   **Location:** Taipei Main Station (Blue Line Transfer Corridor)
*   **Area:** 20 Square Meters (~200 Tiles)
*   **Features:** Integrated power for corridor LED lighting + 2 USB Charging Benches + Digital Dashboard.

## 1. Capital Expenditure (CAPEX)
| Item | Estimated Cost (NTD) | Estimated Cost (USD) | Notes |
| :--- | :--- | :--- | :--- |
| **Energy Tiles (200 Units)** | $2,400,000 | $75,000 | Based on ~$375/tile price point. |
| **Power Management System** | $320,000 | $10,000 | Inverters, batteries, and controllers. |
| **Output Hardware** | $160,000 | $5,000 | 2x USB benches + 1x Large LED Dashboard. |
| **Shipping & Logistics** | $100,000 | $3,125 | International freight and local transport. |
| **Total CAPEX** | **$2,980,000** | **$93,125** | |

## 2. Implementation Costs (OpEx/One-time)
| Item | Estimated Cost (NTD) | Estimated Cost (USD) | Notes |
| :--- | :--- | :--- | :--- |
| **Site Preparation** | $150,000 | $4,680 | Floor excavation and leveling. |
| **Installation Labor** | $250,000 | $7,800 | Specialized technical team. |
| **Software Integration** | $120,000 | $3,750 | Dashboard UI and data tracking. |
| **Total Installation** | **$520,000** | **$16,230** | |

## 3. Total Estimated Budget
*   **Total Initial Investment:** **NT$3,500,000 (Approx. US$110,000)**
*   **Annual Maintenance (Est.):** **NT$50,000** (Cleaning and mechanical checks).

## ROI & Insight
*   **Energy Generation:** At 300,000 steps/day (peak corridor), the system could generate ~1.5 - 2.0 kWh per day.
*   **Environmental Impact:** ~300kg of CO2 reduction per year.
*   **Payback Period:** Strictly from electricity savings, the payback is over 50 years.
*   **Strategic Value:** The primary "Profit" is in **Citizen Engagement** and **Smart City Branding**. A single project like this at Taipei Main Station would generate significant domestic and international media coverage for Taiwan's green tech commitment.

---
*Estimates are based on 2024 industry averages and may vary depending on local vendor availability and station-specific structural requirements.*
